# Pama Lodge Frontend

React + Vite + Tailwind CSS frontend for Pama Lodge booking system.

## Setup

1. Install dependencies:
```bash
npm install
```

2. Start the development server:
```bash
npm run dev
```

The app will be available at `http://localhost:5173`

## Building for Production

```bash
npm run build
```

The built files will be in the `dist` directory.

## Features

- User authentication (Login/Logout)
- Role-based access control (Receptionist/Manager)
- Booking creation and editing
- Booking list with filtering
- Booking detail view with version history (Manager only)
- Daily totals dashboard
- Responsive design with Tailwind CSS
